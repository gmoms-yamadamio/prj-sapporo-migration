# 会員ポイント CSV 作成方針

> 全体方針: [作業方針設計書.md](../作業方針設計書.md)  
> 項目定義: [【サッポロビール様】CSVファイルIF設計書.xlsx](../【サッポロビール様】CSVファイルIF設計書.xlsx) シート「会員ポイント CSV 項目」

## 概要

| 項目 | 内容 |
|---|---|
| 成果物 | 会員ポイント CSV |
| 配置先 | `output/processed/` |
| 投入先 | 移行バッチ **SB00091** |
| 作成方法 | スクリプト（付与・加算） |

## 入力データ

| データ | 配置先 | 役割 |
|---|---|---|
| `User.csv` | `input/raw/` | 会員の基本情報（メールアドレス等） |
| `UserAccount.csv` | `input/raw/` | 会員アカウント（`UserNo` 等） |
| `PointBankAccount.csv` | `input/raw/` | 旧サイトのポイント残高（`UserNo`, `ActivePoint`, `TemporaryPoint` 等） |

## データの結合イメージ

```
UserAccount (UserNo)
    ├─→ User.csv              … 会員メールアドレスの取得
    └─→ PointBankAccount.csv  … シュパークポイント残高の取得
```

### ポイント合算ルール（確定）

`付与ポイント数 = ActivePoint + TemporaryPoint`（整数合算）。合計が 0 以下の会員は出力しない。

## 生成ルール

会員ポイント CSV は、IF 設計書に従い **付与区分 `0`（付与）** の行として出力する。  
旧サイト `PointBankAccount.csv` に記録されているシュパークポイントを、CEC へ付与する。

| 条件 | 処理 |
|---|---|
| **赤星商店に未登録の会員** | 旧サイトのポイント残高を付与として登録する |
| **赤星商店に登録済みの会員** | 赤星商店で登録済みのポイント残高は維持し、**シュパークポイントを加算**する |
| **削除済み会員**（`User.DeletedAt` あり） | **移行対象外** |

### 補足

- 赤星商店登録済み会員に対しては、既存ポイントを上書き・差し替えしない
- 旧サイト側のポイント残高を **付与（加算）** として投入することで、CEC 上の合計ポイント = 赤星商店既存ポイント + シュパーク移行ポイント となる想定
- ポイント残高が 0 の会員（`ActivePoint` + `TemporaryPoint` = 0）は、会員ポイント CSV の出力対象外

## 固定値（IF 設計書より）

| 項目 | 値 |
|---|---|
| 付与区分 | `0`（付与） |
| ポイントキャンペーンコード | `migration-shupark`（固定） |
| サイトコード | `shupark`（固定） |

## 会員 CSV との連携

| 会員 CSV のパターン | 会員ポイント CSV の会員 ID 欄 |
|---|---|
| A. 新規登録（`C`） | 旧サイト `User.csv` のメールアドレス（暫定） |
| B. 更新のみ（`U`） | 赤星商店 CEC 側のメールアドレス（暫定） |

- 会員 ID 欄は IF 設計書どおり暫定でメールアドレスを設定し、**SB00091** が CEC 会員 ID に変換する
- SB00091 実行時には、会員 CSV 由来の `member.csv`（メールアドレス ↔ 会員 ID 対応表）が必要

## 処理順序

会員 CSV の突合・振り分けが完了した後に作成する。

## 項目マッピング

結合: `UserAccount.UserNo` → `PointBankAccount.UserNo` → `User`（[join-keys.md](../../csv-interface/join-keys.md)）

| 会員ポイント CSV 項目 | 必須 | 入力元 | 変換ルール |
| --- | --- | --- | --- |
| 付与区分 | 必須 | — | 固定 `0`（0:付与） |
| 会員ID | 必須 | 会員 CSV 突合結果 | パターン A: `User.Email` / パターン B: CEC メール（SB00091 が会員 ID に変換） |
| ポイントキャンペーンコード | 必須 | — | 固定 `migration-shupark` |
| サイトコード | 必須 | — | 固定 `shupark`（IF 設計書の設定可能値: `lager`/`shupark`/`eshop`） |
| 付与ポイント数 | 必須※ | `ActivePoint` + `TemporaryPoint` | 合算して 1 行付与（確定）。付与区分が付与のため必須 |
| 消化ポイント数 | — | — | 空（付与区分が消化の場合のみ必須のため未設定） |
| Description | 任意 | — | 空または `shupark migration` |
| メモ | 任意 | — | 未設定（IF 設計書に定義があるが本移行では不要。最大 1024 文字） |
