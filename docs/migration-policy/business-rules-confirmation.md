## お客様への確認依頼文（テンプレート）

以下をサッポロビール様へ送付し、回答を本ドキュメントに反映する。

1. ~~退会済み（削除済み）会員を移行対象から除外してよいか~~ → **移行対象外（確定）**
2. ~~ゲスト注文の注文履歴を移行するか~~ → **移行対象外（確定）**
3. ~~キャンセル済み注文（orderStatus=10）を移行するか~~ → **移行対象外（確定）**
4. ~~ポイントは ActivePoint と TemporaryPoint を合算してよいか~~ → **合算（確定）**
5. ~~旧会員 ID カスタム項目には `User.Id` を格納してよいか~~ → **`User.Id` を格納（確定）**
6. ~~旧 siteId `1` / `3` の CEC サイトコード対応~~ → **一律 `shupark`（確定）**
7. ~~商品マスタコードのプレフィックスは `mig_` か `shupark_` か~~ → **`mig_`（確定）**
8. ~~酒類フラグ（[カスタム]酒類フラグ）はどの情報から判定すればよいか~~ → **全商品一律 `対象外`（確定）**
9. ~~移行した商品は登録時点で「販売停止」「非表示」を `1`（停止・非表示）にしてよいか~~ → **一律 `1`（販売停止・非表示）で移行（確定）**。OPEN 後の販売商品は別途登録
10. ~~配送温度帯IDの固定値は `A1111` か `T1111` か~~ → **`T1111`（確定）**
11. ~~決済方法グループコード／発送元コード／配送方法グループコードに設定する実際の値~~ → **`migration-pay-group` / `migration-warehouse` / `migration-delivery-group`（確定）**
12. 注文履歴CSVの調整金額（`discountPrice`）は旧データ上プラス値かマイナス値か → **移行データ調査後に判断（未決）**
13. ~~軽減税率対象商品（8%）の税率をどの列で受領するか~~ → **SKU 一覧共有で確定**（#21 / #27）。指定 12 SKU が `8`、それ以外 `10`
14. 注文履歴CSVの受取人メールアドレスはブランク（空値）で出力してよいか → **暫定方針はブランク出力。お客様合意が必要（#28）**
15. ~~`[カスタム]法人コード`は新規会員登録時に必須項目とされているが、法人会員以外（個人会員）の移行時は空値で登録してよいか~~ → **一律ブランクで登録（確定）**。CSV アップロード時にエラーとならないか、念のためお客様側にも申し送りを推奨
