# サイトコード / 管理グループコード（確定）

> 版: 確定  
> 更新日: 2026-07-01

## 方針

**移行元データはすべてシュパーク 1 サイトのデータ**である。  
旧サイトの `siteId` 列に `1` / `3` 等の値が混在していても、CEC への出力では **サイトコード・管理グループコードを一律 `shupark`** とする。

## 対応表

| 旧 `siteId` | CEC サイトコード | 管理グループコード | 備考 |
| --- | --- | --- | --- |
| すべて | `shupark` | `shupark` | 旧 siteId は参照しない |

## 成果物ごとの適用

| 成果物 | サイトコード | 管理グループコード |
| --- | --- | --- |
| 注文履歴 CSV | `shupark` | `shupark` |
| 会員ポイント CSV | `shupark` | — |
| 商品マスタ / 販売基本情報 / 販売条件 | — | `shupark` |
