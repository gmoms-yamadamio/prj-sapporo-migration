# 赤星商店 CEC 会員データ（`input/cec/`）

## 目的

旧サイト会員とクラウド EC 既存会員（赤星商店）の突合に使用する。

## 取得方法（確定）

**CEC 管理画面から会員 CSV をエクスポート**し、本フォルダに配置する。

1. CEC 管理画面 → 会員管理 → 会員一覧
2. エクスポート（全件または赤星商店会員）
3. ファイルを `member_export.csv` として保存
4. 本番移行時は会員インポート後に `member.csv`（SB00091/92 用）も同フォルダに配置

## 突合に使用する列

| 列名（CEC エクスポート） | 用途 |
| --- | --- |
| 会員ID | パターン B（更新 U）の会員 ID 指定 |
| メールアドレス | **突合キー（第一候補）** |

## SB00091/92 用 member.csv フォーマット

会員インポート完了後、以下形式でエクスポートまたは整形する:

```csv
会員ID,メールアドレス
12345,user@example.com
```

- 配置先: `input/cec/member.csv`
- SB00091: メールアドレス → 会員 ID 変換に使用
- SB00092: 注文の会員紐付けに使用

## サンプルファイル

`member_export_sample.csv` はフォーマット確認用のダミーデータ（マスク済みメール）。  
実際の突合検証には本番相当の CEC エクスポートに差し替えること。
